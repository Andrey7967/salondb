5.2. Выбор технологий
Для достижения указанных требований было решено использовать Electron. Он обеспечивает кроссплатформенность, быстродействие, взаимодействие с операционной системой и использует актуальные технологии. 
5.4. Обоснование выбранного метода
Веб-технологии: Выбор HTML, CSS и JavaScript для создания пользовательского интерфейса позволил использовать известные технологии и легко интегрировать клиентский интерфейс.
Node.js: Асинхронная модель Node.js и возможность работы с файловой системой обеспечили высокую производительность и надежность в обработке данных и взаимодействии с базой данных.
